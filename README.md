# Client-Server-JScript-TCP-Communication
TCP communication between Client and Server based on socket programming using JavaScript and NodeJS

## Usage
U have to install nodejs. After u are set with nodejs then just go to cmd and change directory to folder where u extracted these files.
type `node server.js` in one cmd and in the next one type `node client.js`. Now u have made the connection next just follow code instructions.
